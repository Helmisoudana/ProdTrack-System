# ProdTrack-System
**ProdTack** est un système IoT et Web pour le suivi de production (textile). Il utilise des boîtiers RFID (ESP32) sur les machines pour le pointage en temps réel des opérateurs. Le backend Python (FastAPI) gère ces données et déclenche des alertes d'anomalies grâce au Machine Learning. Le manager visualise tout via un tableau de bord Next.js.
